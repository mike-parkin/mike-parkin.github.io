Positioning in CSS: relative, absolute and fixed

Before discussing position attributes in CSS, it is best to start by mentioning that everything displayed on webpage is made up of elements. Some elements are placed inside other elements, like a paragraph inside an article for example. All elements are rectangular in shape and start with a default position of ‘static’. This places the element within the ‘flow’ of the document, that is either, beneath, or to the right of the previous element. By using CSS we can manipulate the element using the ‘position’ property. Here I will be discussing three types of positioning and how they work. These are: relative, absolute, and fixed.

Relative

Relative positioning places the object relative to itself, which is confusing, just setting the position as relative will not change anything but by adding positioning properties (top, right, bottom, left), the object will position itself relative to the its position in the flow of the document, in simple terms; the top left corner of the element.

As an example, In this paragraph you will see some words are out of line with the rest of the text. This text has been placed within span tags and their relative position has been altered.

Look at the word “some”. it has been placed within its own element inside the paragraph and that element’s position is set to relative, and then the bottom position has been set to 3px, so it has moved the text up 3px from where it would normally be. The code below shows how the word 'some' is moved of its line using relative position.


                HTML: <span class="up">some</span> 

                CSS: 
                .up {
                position: relative;
                bottom: 3px;
                }

Absolute

An element with absolute position will be lifted out of the “flow” of the document, meaning it doesn’t stay in position with the other elements around it, and can be positioned exactly where you want it on the page using top, right, bottom and left attributes. An element with the position of absolute will only be positioned relative to the body of the page, OR if it’s placed within another element with a position attribute of anything but static, it will be positioned relative to that parent element.

The small yellow box in the top right is set to an absolute position of 10px from the top, in relation to the body of the document, the small red box in the top corner of this article is also set to an absolute position of 10px from the top, but it is inside an element with relative position, there for the red box is 10px from the top if this parent element.


                .red-box {
                    position: absolute;
                    top: 10px;
                    left: 10px;
                    height: 20px;
                    width: 20px;
                    background-color: red;
                }

Fixed

Fixed position places the element on the page relative to the body and viewport (screen) of the page, This element will stay in the exact same place even when the page is scrolled, this is useful if you want to have anchors or menus always visible to the user. For example the blue box at the bottom rigth of this page has been put in fixed position so no matter how far down this blog you scroll, the blue box will always be in the same postion on the screen

 
                .blue-box {
                    position: fixed;
                    bottom: 10px;
                    right: 10px;
                    height: 20px;
                    width: 20px;
                    background-color: blue;
                }
